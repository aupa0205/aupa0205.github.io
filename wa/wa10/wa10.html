Random Gaming Trivia

Extension: For my extension, I turned the Christmas trivia machine into a random gaming trivia generator. Each time you click for a new question, it pulls a fresh piece of gaming trivia and lets you reveal the correct answer. I also added a changing background color on the buttons to keep things visually interesting.

API in critique project: I could imagine using an API in my critique project to pull in real-world gaming data, like statistics, player feedback, or quotes from developers, to make the content more dynamic and relevant. Using live data would let the project evolve over time rather than staying static.
